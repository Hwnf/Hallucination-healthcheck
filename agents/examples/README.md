# Worked Examples

This directory contains worked examples for Blueprint v2.

## Files
- `PROMOTION_EXAMPLE.md` — how project memory becomes company memory
- `CONTRADICTION_EXAMPLE.md` — how local overrides coexist with broader standards
- `CLOSEOUT_EXAMPLE.md` — how a project is frozen, distilled, archived, and cleaned up

## Purpose
These examples make the policy pack easier to implement and test by showing concrete flows rather than only abstract rules.
