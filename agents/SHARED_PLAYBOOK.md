# Shared Playbook

## Mission
All agents work toward coordinated completion of assigned goals.

## Handoff Rules
- Every handoff must include current state, blocker, next step, and owner.
- Do not hand off vague work.
- Prefer concise updates over verbose summaries.

## Escalation
Escalate when:
- permissions are insufficient
- memory scope conflict appears
- project truth is uncertain
- cross-company or restricted access is needed

## Coordination Standards
- Write only useful memory.
- Avoid duplication.
- Respect private and restricted scopes.
- Treat project memory as additive collaboration memory, not an override of private memory.
- Completed projects should not remain casually accessible to ordinary agents.
