# Plugin Type Skeleton

This directory contains the initial TypeScript interface and type skeleton for the memory policy plugin.

## Included
- `types/` — core contracts and enums
- `interfaces.ts` — service/module interfaces

## Purpose
This is the implementation bridge between:
- Blueprint v2 canon
- policy docs
- registry structures
- actual plugin code
